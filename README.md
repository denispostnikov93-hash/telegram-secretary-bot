# Telegram Бот-Секретарь 🤖

Автоматизированный бот для приема заявок на консультации через Telegram с поддержкой GDPR и согласий на обработку персональных данных.

## Функции

- ✅ Полный цикл приема заявок через Telegram
- 📋 Двух-уровневая система согласий (GDPR compliant)
- 👤 Поддержка физических и юридических лиц
- 📂 Категоризация юридических вопросов
- 💾 Сохранение заявок в SQLite базе
- 📱 Отправка уведомлений в рабочий чат
- ✔️ Валидация номеров телефонов
- 🔐 Логирование всех операций

## Требования

- Python 3.8+
- Telegram Bot API Token
- ID админ-чата Telegram

## Установка

1. Клонируйте репозиторий:
```bash
git clone <repo-url>
cd telegram_secretary_bot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте `.env` файл на основе `.env.example`:
```bash
cp .env.example .env
```

4. Заполните переменные окружения:
```
TELEGRAM_BOT_TOKEN=ваш_токен_бота
TELEGRAM_ADMIN_ID=ваш_id
TELEGRAM_CHAT_ID=id_рабочего_чата
```

5. Запустите бота:
```bash
python main.py
```

## Структура файлов

- `main.py` - точка входа
- `telegram_bot.py` - основная логика бота
- `config.py` - конфигурация
- `database.py` - работа с БД
- `requirements.txt` - зависимости

## Поток взаимодействия

1. Пользователь нажимает /start
2. Требуется подтверждение двух согласий (политика + обработка данных)
3. Выбор типа клиента (физ лицо / юр лицо)
4. Выбор категории вопроса
5. Ввод имени и телефона
6. Опциональное описание ситуации
7. Заявка сохраняется в БД и отправляется в рабочий чат
