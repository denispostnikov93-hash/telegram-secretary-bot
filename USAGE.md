# 📱 Бот-секретарь Telegram - Инструкция

## 🚀 Как это работает при открытии бота

### Шаг 1: Первое открытие
Когда пользователь открывает бота в Telegram, он видит:
- **Встроенная кнопка "Start"** (это встроенная функция Telegram)
- После нажатия на неё Telegram отправляет команду `/start`

### Шаг 2: После нажатия Start
Бот показывает:
```
👋 Добро пожаловать в Правовой центр "Постников групп"!

Мы поможем защитить ваши права.

📋 ПРОЦЕСС ЗАПИСИ НА КОНСУЛЬТАЦИЮ:
1️⃣ Согласие на обработку данных
2️⃣ Выбор типа клиента (физлицо/юрлицо)
3️⃣ Категория вопроса
4️⃣ Ваше имя и телефон
5️⃣ Краткое описание ситуации
6️⃣ Получение консультации

Нажмите кнопку ниже или напишите любое сообщение для начала:
```

**Кнопка:** 📝 Записаться на консультацию

---

## 💡 Две способа запуска процесса

### Способ 1️⃣: Нажать кнопку
- Пользователь нажимает кнопку "📝 Записаться на консультацию"
- Сразу начинается цикл согласий

### Способ 2️⃣: Написать любое сообщение
- Пользователь просто пишет что-угодно в чат
- Бот автоматически запускает цикл записи
- **Это работает только в главном меню** (когда нет активной формы)

---

## 📋 Полный цикл приема заявки

1. ✅ Согласие на обработку персональных данных
2. ✅ Согласие с политикой обработки данных
3. 👤 Выбор типа клиента:
   - 👤 Физическое лицо
   - 🏢 Юридическое лицо
4. 📂 Выбор категории (разные для физлиц и юрлиц)
5. 📝 Ввод имени
6. 📱 Ввод телефона (с валидацией)
7. ❓ Выбор: написать описание или пропустить
8. (опционально) Ввод описания ситуации
9. ✅ Заявка отправляется в рабочий чат

---

## 🔄 Автоматический запуск

**Если пользователь напишет сообщение в главном меню:**
- Сообщение будет проигнорировано
- Бот автоматически предложит начать процесс записи
- Это работает **только** если пользователь не в процессе заполнения формы

---

## ⚙️ Переменные окружения (.env)

Убедитесь, что установлены:
```
TELEGRAM_BOT_TOKEN=<ваш токен от BotFather>
TELEGRAM_ADMIN_ID=<ваш ID в Telegram>
TELEGRAM_CHAT_ID=<ID рабочего чата для заявок>
```

---

## ✅ Проверка работы

1. Запустите бота: `python main.py`
2. Откройте бота в Telegram
3. Нажмите кнопку "Start"
4. Вы должны увидеть приветствие со списком этапов
5. Нажмите кнопку "📝 Записаться на консультацию" или напишите любое сообщение

---

## 📊 Заявки

Все заявки:
- Сохраняются в БД (`applications.db`)
- Отправляются в рабочий чат (Telegram)
- Содержат всю информацию (имя, телефон, тип, категорию, описание, согласия)
